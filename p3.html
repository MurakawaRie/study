5. 다음 Classful 문제에 대해 서술하세요
    5-1. 아이피 192.168.10.100
  • Class : Class C
    Subnet Mask : 255.255.255.0
    Network 주소 : 192.168.10.0
    할당 가능한 IP 주소 : 192.168.10.0 ~ 192.168.10.254
    Broadcast 주소 : 192.168.10.255
    5-2. 아이피 2.1.1.250
  • Class : Class A
    Subnet Mask : 255.0.0.0
    Network 주소 : 2.0.0.0
    할당 가능한 IP 주소 : 2.0.0.0 ~ 2.255.255.254
    Broadcast 주소 : 2.255.255.255
    5-3. 아이피 154.123.67.8
  • Class : Class B
    Subnet Mask : 255.255.0.0
    Network 주소 : 154.123.0.0
    할당 가능한 IP 주소 : 154.123.0.0 ~ 154.123.255.254
    Broadcast 주소 : 154.123.255.255
6. linux의 명령어에 대한 질문에 서술하세요
    6-1. 파일의 내용을 출력하는 명령어
  • -cat : 전체 내용 불러오기(-n을 붙이면 n 만큼 열 만큼 출력)
    *파일명 없이 cat 사용시 모니터에 입력된 내용을 바로 출력

    -head : 위쪽부터 내용 불러오기(-n을 붙이면 n 만큼 열 만큼 출력)
    -tail : 아래쪽부터 내용 불러오기(-n을 붙이면 n 만큼 열 만큼 출력)
    -more : 화면 사이즈에 맞게 내용을 출력
    6-2. 최고 권한으로 실행시켜주는 명령어
  • sudo [실행할명령] : root 권한으로 명령어를 실행시켜줌.
    6-3. 파일 혹은 디렉토리의 이름을 변경하거나 다른 위치로 이동시키는 명령어
  • mv [원본경로] [대상경로] : 대상경로 입력시 파일명을 변경 할 수 있음
    6-4. 파일 혹은 디렉토리의 소유자/소유그룹을 변경하는 명령어
  • chown (옵션) 소유자명 파일/폴더명 : 해당 폴더/파일의 소유자를 변경
    *옵션에 -r을 붙이면 하위파일과 디렉토리까지 포함하여 변경한다

    chown 소유자명 폴더명: 소유자명을 바꿀 수 있음
    chown .소유그룹명 폴더명 : 소유그룹명만 바꿀 수 있음
    6-5. 파일 혹은 디렉토리의 접근 권한을 변경하는 명령어
  • chmod 권한 파일명 : 파일에 대한 특정 권한 부여
    * r(읽기 ls,cat) / w(쓰기 touch) / x(실행 및 접근하기 cd)
    * u유저 / g그룹 / o외부인 / a전체 // +기존에추가 / -삭제 / =입력된것만 변경

    예시
    chmod u+rw filename(유저에게 filename에 대한 r,w 권한 추가)
    chmod g-rw filename(그룹에게 filename에 대한 r,w 권한 삭제)
    chmod a=rx filename(전체에게 filename에 대한 r,x 권한 변경)
    chmod u=rwx,g=r,o=r(유저에게 r,w,x권한 변경 / 그룹과 외부인에게 r권한 변경)
  • 2진법을 활용하여 chmod를 사용 가능하다
    파일/디렉토리에서 ls -l 입력시 rw-r--r-- 같은 형태로 출력되는게 권한인데
    3개씩 나뉘어 유저,그룹,외부인 권한을 표현한다
    --x 1 / -w- 2 / r-- 4
    예시
    chmod 755 -> 755(rw- , r-x , -r-x)
    6-6. 디렉토리를 만들기 위해 사용하는 명령어
  • mkdir (-p) : -p를 붙이면 상위 폴더도 포함하여 생성
    6-7. 파일 혹은 디렉토리를 다른 위치로 복사하는 명령어
  • cp [옵션-생략가능][원본파일경로][대상파일경로] (-r) : -r을 붙이면 디렉토리를 포함하여 복사가능
    6-8. 경로 이동에 사용하는 명령어
  • cd 절대경로(경로를 직접 입력하여 이동)
    cd 상대경로(현재 경로 기준에서 이동)
    . 현재폴더 / .. 하위폴더 / ~ 홈디렉토리
