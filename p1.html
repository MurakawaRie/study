연습문제 part 1

1. 웹 어플리케이션은 독립 어플리케이션에 비해 웹 기반 응용프로그램으로서
웹 브라우저 상에서 HTTP 프로토콜 및 HTML 문서를 근간으로 만들어지는 어플리케이션이다
    1-1. 웹 어플리케이션엔 어떠한 것이 있는지 예시 3개를 들어보세요
  • 웹브라우저(Edge,Chrome,Firefox 등)
  • 웹 서버(Apache Server, IIS 등)
  • 웹 어플리케이션 서버(Apache Tomcat 등)
    1-2. 웹 사이트와 웹 어플리케이션의 차이에 대해 서술하세요
  • 웹 사이트(Website)는 정보 관람을 목적으로 하는 정적인 "사이트"
