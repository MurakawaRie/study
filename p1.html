연습문제 part 1

1. 웹 어플리케이션은 독립 어플리케이션에 비해 웹 기반 응용프로그램으로서
웹 브라우저 상에서 HTTP 프로토콜 및 HTML 문서를 근간으로 만들어지는 어플리케이션이다
    1-1. 웹 사이트에 대한 정의
  • 웹 사이트(Website)는 정보 관람을 목적으로 하는 정적인 "사이트"
    대표적인 예시로는 뉴스페이지나 문서 페이지가 있다
    1-2. 웹 어플리케이션의 정의와 구성요소
  • 웹 어플리케이션(application)은 동적인 기능을 가진 웹
    유저간의 상호작용=소통을 가능하게 하여 유저가 능동적으로 서비스를 이용가능
    대표적인 예시로는 지도 어플이 있다.


  • 웹 어플리케이션의 구성요소에는 총 4가지가 있다
    - 웹 브라우저 : 사용자가 요청한 웹 서버의 결과를 화면에 출력해서 보여주는 어플리케이션
    (대표적으로 Edge,Chrome,Firefox 등의 인터넷 브라우저들)

    - 웹 서버 : 웹 브라우저의 요청을 받아 그 결과를 브라우저로 응답해주는 역할을 하는 어플리케이션
    (대표적으로 Apache Server, IIS 등이 있다)

    - 웹 어플리케이션 서버 : 요청한 페이지의 로직 또는 데이터베이스와의 연동을 처리하는 부분
    (대표적으로 Apache Tomcat 등이 있다)

    - 데이터베이스 : 데이터를 저장하는 곳이다
    (대표적으로 : MySQL, Oracle 등을 사용)

3. 다음 java코드를 분석하고 틀린 부분이 있을시 수정하세요
그리고 각각의 변수타입비트단위도 서술하세요.
    3-1. float a = 123.2f;
  • 문제없음, float는 4byte 형태의 실수를 표현.
    3-2. double b = 123.2;
  • 문제없음, double는 8byte 형태의 실수를 표현.
    3-3. boolean c = 1;
  • boolean는 true false로 선언 가능한 논리형. 1byte로 표현
    ※원시타입과 참조타입
  • 원시타입(Primitive Type)
    정수,실수,문자,논리 리터럴 등 실제 데이터 값을 저장하는 타입
  • 참조타입(Reference Type)
    원시 타입을 제외한 타입(문자열,배열,열거,클래스,인터페이스)를 말함
