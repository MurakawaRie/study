1. 다음 질문에 맞는 답을 서술하세요
    1-1. MVC는 무엇의 약자인지 서술하고 각각의 역할과 요소를 서술하세요

  • M : Model

    어플리케이션의 정보와 데이터를 나타냄
    데이터베이스, 초기화값, 변수 등을 뜻함

    V : View

    input텍스트, 체크박스 등 사용자 인터페이스 요소를 나타냄
    데이터 및 객체의 입력, 출력을 담당하며 사용자가 볼 수 있는 화면

    C : Controller

    데이터와 사용자 인터페이스 요소를 잇는 다리역할을 함
    사용자가 데이터를 클릭,수정하는 등 "이벤트"를 처리하는 부분
2. 다음 코드를 보고 질문에 답하세요
    2-1.
    public class Test{}
    public class Test extends Object{}
    두 코드는 같은 코드입니까? 왜 그런지도 서술하세요
  • 같은코드이다
    모든 자바의 클래스는 Object를 암묵적으로 상속받고 있어 생략되어있기 때문
    2-2.
    public class Test extends Object {
    super();
    }
    다음 코드는 맞는 코드입니까?
  • 아니다
    모든 클래스는 Object를 암묵적으로 상속 받고 있으므로 super()를 쓰면 다중상속이다
    디폴트 생성자 안에 넣어주어야 한다
3. 다음 설명에 대한 질문에 답하세요
    3-1.
    1) 이것은 특정 객체와 데이터 자료형이 가지고 있는 함수다
    2) 데이터와 멤버 변수에 대한 접근 권한을 가지고 있다
    3) 특정한 작업이나 논리를 구성하는 코드를 괄호로 묶은 것이다
    4) 입력 값을 받아서 내부에서 처리해 결과를 출력 및 반환 한다
  • 메소드(Method)
    3번의 괄호는 중괄호로 묶는것을 말한다
    3-2. 메소드를 호출하는 방법에 대해 서술하세요
  • void 함수명() {return 선언 없음}
    변수타입 함수명() {변수타입에 맞는 return 선언 있음}
    3-3. 메소드는 이것을 만나면 값을 반환하지 않고 종료된다
