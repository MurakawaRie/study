3. 다음 CSS 코드들에 대해 설명하세요
    2-1. display : flex
  • 요소들의 width 만큼만 차지하면서 가로방향으로 배치시킴
    height는 요소의 최대 높이만큼 알아서 늘어남
    2-2. opacity
  • 투명도 조절(0~1)
    2-3. border
  • 테두리를 만들어 주는 코드
    보통 border : *px solid color순서로 사용한다
    (*는 두께, solid는 테두리 타입, color는 컬러)
    2-4. z-index
  • 겹쳐진 요소들의 위아래 우선순위를 지정할 수 있다
    숫자가 클 수록 우선순위가 위라서 맨 위에 올라오게 됨
    2-5. align-items
  • 소속된 요소 안에 포함된 모든 요소들을 소속된 요소의 높이에 맞게 세로정렬해줌
    align-items : start(시작지점 기준)
    align-items : end(끝지점 기준)
    align-items : center(중간지점으로)
    참고용 링크
    2-6. justify-content
  • 메인축을 기준으로해서 요소들을 정렬하는것
    space-between : 요소들의 사이(between)에 균일한 간격을 만듬
    space-around : 요소들의 둘레(around)에 균일한 간격을 만듬
    space-evenly : 요소들의 사이와 양 끝에 균일한 간격을 만듬
    2-7. padding
  • 내용(content)과 테두리(border)사이의 간격(padding)을 설정한다
    (예제)
    padding : Apx Bpx Cpx Dpx; 아랫줄과 동일한 코드임
    padding-top:Apx; padding-right:Bpx; padding-bottom:Cpx; padding-left:Dpx;
    top,right,bottom,left 순서로 설정가능
    2-8. margin
  • 테두리(border)와 이웃하는 요소 사이의 간격(margin)을 설정한다
    위의 padding과 동일하지만 음수를 사용해 다른요소랑 겹치게 가능
    2-9. transition
  • CSS가 특정 상황에서 변화할때 일정시간에 걸쳐 일어나게 해주도록 하는것
    (예제) hover시 특정 효과가 발동되게 했을때
    transition : all 2s;(모든 변화를 2초에 걸쳐서 변화하게 함)
    참고용 링크
    2-10. padding과 margin의 차이?

  • 위의 그림 참고
    padding은 요소와 border 사이의 간격이고
    margin은 다른요소와 border 사이의 간격이다
